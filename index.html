How to Use Adeptus Minimus

1. Upload a ListForge JSON
Use “Initiate Data Uplink” to load a roster exported from ListForge. Or use the integration within the Listforge app.

2. Toggle Descriptions
“Expose Descriptions” shows or hides ability and enhancement text.

3. Generate a PDF
“Forge PDF Cards” exports all rendered cards as a printable PDF.
